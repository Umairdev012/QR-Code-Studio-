# QR Code Studio

A clean and modern QR Code Generator built using **HTML, CSS, and Vanilla JavaScript**. Users can instantly generate QR codes from any text or URL, download the generated QR code, and clear the form with a single click.

## ✨ Features

- 🔳 Generate QR codes from text or URLs
- ⌨️ Press **Enter** to generate instantly
- 📥 Download generated QR code as PNG
- 🗑️ Clear input and reset generator
- 📱 Fully responsive design
- 🎨 Modern light UI with smooth animations
- ⚡ Fast and lightweight
- 🚫 No frameworks required

## 🛠️ Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript
- QR Code API

## 🚀 Getting Started

1. Download or clone this repository.
2. Open the project folder.
3. Open `index.html` in your browser.
4. Enter any text or URL.
5. Click **Generate** to create a QR code.
6. Click **Download** to save the QR code.
